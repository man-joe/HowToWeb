How to... Dual Boot Windows and Linux Mint

by Joe
There are many benefits to having multiple operating systems available on your computer. This guide will teach you how to install the most current versions of Windows and Linux Mint on a computer that already has Windows 10 installed. Make sure you have an 8GB USB available that you don't mind having wiped.
